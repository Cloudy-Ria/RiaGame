Сцены должны называться Имя_Число, к примеру, Hometown_01.
Сцены с одинаковым именем должны быть похожими по смыслу и примерному размеру, но разными по исполнению для симуляции плохой памяти персонажа.

Комнат с одинаковым именем может быть сколько угодно, но их числа должны быть все по порядку от 01 до XX.

При добавлении новой сцены важно закинуть ее в список сцен в File > Build Profiles > Scene List

Для планирования порядка, имен, предназначений и наполнений комнат используется Фигма, https://www.figma.com/design/7Slb9UULDPb2jukSRNBHgt/Ria-Game-Concept?node-id=0-1&t=3fQJDp8Ri9RlOPqJ-1

P.s. Используй Box Collider 2d в обьекте Scene Boundary для ограничения движения камеры по сцене.
